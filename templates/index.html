DevOps là gì?

DevOps là sự kết hợp giữa Development (Phát triển) và Operations (Vận hành). Đây là một phương pháp làm việc nhằm cải thiện sự hợp tác giữa các nhóm phát triển phần mềm và vận hành hệ thống.

CI - Continuous Integration (Tích hợp liên tục)

CI là quy trình tự động hóa việc tích hợp code từ nhiều nhà phát triển vào một kho lưu trữ chung. Mỗi khi có thay đổi, hệ thống sẽ tự động build và test để phát hiện lỗi sớm.

  • Tự động build code
  • Chạy unit test tự động
  • Phát hiện lỗi sớm

CD - Continuous Delivery/Deployment (Triển khai liên tục)

CD là quá trình tự động hóa việc triển khai ứng dụng lên môi trường production. Giúp đưa sản phẩm đến tay người dùng nhanh chóng và an toàn hơn.

  • Tự động deploy lên server
  • Giảm thiểu lỗi khi triển khai
  • Phát hành phần mềm nhanh chóng
